### 2.2 参数说明

| 参数名称         | 是否必须 | 默认值 | 描述                                                            | 
|--------------|------|------|---------------------------------------------------------------|
| host         | 是    | 127.0.0.1 | 数据库地址                                                         |
| port         | 是    | 3306 | 数据库端口                                                         |
| db           | 是    |  | 数据库名称                                                         |
| username     | 是    | root | 数据库用户名                                                        |
| password     | 是    | 123456 | 数据库密码                                                         |
| query        | 是    |  | 查询语句                                                          |
| mode           | 否    | batch | 写入方式，支持batch、cursor模式,batch则直接一次性从数据库获取所有数据，<br/>cursor则一次从数据库获取1条 |
